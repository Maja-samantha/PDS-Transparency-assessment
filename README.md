# PDS-Transparency-assessment
The code is used for data analysis of the data extraction from the journal, Pharmacoepidemiology and Drug Safety (PDS), with the purpose of performing a transparency assessment across 5 transparency domains. The data was extracted using the software Silvi.ai.
The code is guided, with explanations and descriptions written in.
The code may be used by other coders as they see fit, for it to work on other journals where a transparency assessment is being performed. 
Remember to look at fallback for the data entries that do not match any categories, and rewrite the code accordingly.
